## Intro to PyEPICS
## Installation Dependencies: * EPICS Base shared libraries * Python 2.6+ or Python 3.2+ * *Optional:* numpy for nicer array handling ```bash pip install pyepics export PYEPICS_LIBCA=/epics/base/lib/linux-x86_64/libca.so ```
## Getting values ```python >>> import epics >>> >>> beam_current_pv = epics.PV('SR11BCM01:CURRENT_MONITOR') >>> beam_current_pv.get() 165.304845902 >>> >>> beam_mode_enum_pv = epics.PV('FS01:BEAM_MODE_MONITOR') >>> beam_mode_enum_pv.get() 2 >>> beam_mode_enum_pv.get(as_string=True) 'Maintenance' ```
